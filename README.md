# 🎮 Quiz Challenge

Quiz Challenge is a Python command-line quiz game that tests players on technology and general knowledge.  
It includes difficulty levels, scoring, lives, streak bonuses, and a leaderboard system using file storage.



*                    Features

- 🎯 Multiple-choice questions
- 📂 Questions loaded from a JSON file
- 🧠 Difficulty levels (Easy, Medium, Hard)
- ❤️ Lives system (3 chances per game)
- 🔥 Streak bonus points system
- 💯 Score tracking
- 🏆 Player ranking system
- 💾 Saves scores to a file
- 🔄 Play again option



* Technologies Used

- Python 3
- JSON (for storing questions)
- File handling (TXT leaderboard)
- Random module
